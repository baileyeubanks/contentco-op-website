This folder is the canonical full-master portfolio bank for CCO curation work.

Rules:
- Only full `.mp4` / `.mov` masters belong here.
- `preview_*` files are hover/loading assets only and are intentionally excluded.
- `thumb_*` files are thumbnails only and are intentionally excluded.
- `current_public_master` files are already mapped in the public manifest.
- `candidate_archive_master` files are valid replacement/addition candidates.
- `candidate_instagram_master` files are intentionally kept because they were explicitly confirmed as high-signal candidates.
- `removed_from_public` means the master still exists, but the entry is no longer in the public manifest.
